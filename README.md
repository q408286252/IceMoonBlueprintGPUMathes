### IceMoon Blueprint GPU Math (IMBGM)

** 这是一个UE插件 解决脑淤血的蓝图中关键字缺失的节点：提供一套统一、高性能的 C++ 蓝图节点。**

### 🚀 核心价值：GPU 关键字统一与 C++ 性能

本库致力于解决技术美术 (TA) 和图形开发者在 **Niagara**、**蓝图 (Blueprint)**、**材质 (Material/HLSL)** 和 **C++** 之间切换时，因函数命名和性能不一致导致的心智负担。所有节点均使用 `IM_` 前缀，以确保不与引擎或第三方插件产生冲突。 核心是抹平各环境下的不同关键字问题

**设计理念核心范例：HLSL 语义一致性**

|**HLSL/GLSL 关键字**|**蓝图节点 (DisplayName)**|**统一优势**|
|---|---|---|
|`saturate(x)`|`Saturate (GPU)`|避免使用冗长的 `FMath::Clamp(x, 0.f, 1.f)`，与 GPU 编程中的语义完全一致。|

所有函数都可在蓝图的 `IM | GPU | Math` 和 `IM | GPU | Rotation` 分类中找到。该库的内容会根据作者的**实际项目需求**进行迭代更新，以确保所有函数都具有高实用性。

### 🛠️ 安装

1. 将此仓库克隆或下载到您的 Unreal Engine 项目的 `Plugins` 文件夹中。
    
2. 在 `.uproject` 文件中确保插件被启用。
    
3. 重新生成 Visual Studio 项目文件并编译。
    

### ⚠️ 工程规范与维护政策

本插件旨在作为作者的**个人作品集**和**工作流工具**。请在使用前理解以下限制：

- **性能警告：** 所有 `IM | GPU` 节点都使用 C++ 编写。但任何蓝图节点的频繁调用（例如在 `Event Tick` 中）都会产生 **蓝图解释器的开销**。请优先将高频数学计算置于 **材质或 C++ 代码** 中。
